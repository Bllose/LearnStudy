# 获取资源信息  
## ORACLE  
查看存储过程：
``` SELECT * FROM USER_SOURCE WHERE TYPE = 'PREOCEDURE'; ``` 

查看方法：
``` SELECT * FROM USER_SOURCE WHERE TYPE = 'FUNCTION'; ```  

查看包：
``` SELECT * FROM USER_SOURCE WHERE TYPE LIKE 'PACKAGE%'; ```  

查看队列：
``` SELECT * FROM USER_SEQUENCES; ```
